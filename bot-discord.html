Création du bot discord

Bonjour et bienvenue, dans cet article je vous explique comment utiliser le bot discord Skybot.

Pour commencer ce bot n'est disponible que sur un seul serveur discord, le serveur "Skydia". Pour le rejoindre, rendez vous sur la page Information et cliquez sur le bouton "connect".

Pour commencer, nous vous prions de ne pas utiliser la commande "site" ou la commande "serveur". Les règles figurerons en bas de cette page.

Pour obtenir toutes les commandes du bot tapez ".help", toutes les commande vont alors s'afficher.

Les seules commandes accessible que vous avec le droit d'utilisiser sont :

1) ".addition [nombre1] [nombre2]"

2) ".ping"

3) ".info"

! attention !

Je peux voir toute les logs du bot donc si vous faite n'importe quoi vous ne pourrez plus utiliser le bot.
